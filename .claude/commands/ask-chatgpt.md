# Prepare a decision summary for ChatGPT

Decision context: $ARGUMENTS

Use this command when:

- Phase scope is unclear.
- Codex and Copilot reviews conflict.
- Billing, security, or authentication architecture is uncertain.
- A change may affect future phases.
- A blocking issue is ambiguous.

Do not continue coding or modify files. Prepare a concise summary containing:

1. Current phase
2. Current task
3. Files changed
4. Review feedback
5. Decision needed
6. Options and their tradeoffs
7. Recommendation

End by telling the user to paste the summary into ChatGPT.
